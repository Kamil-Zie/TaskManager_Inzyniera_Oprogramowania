## 1. Backend (TaskManager_API)

Backend jest aplikacją ASP.NET Core, która obsługuje logikę biznesową i komunikację z bazą danych MongoDB.

### Wymagania:
- Środowisko .NET 8.0 SDK
- Działająca instancja MongoDB (domyślnie na `mongodb://localhost:27017`)

## 2. Frontend (taskmanager-webapp)

Frontend jest aplikacją React, która komunikuje się z backendem, aby wyświetlać i zarządzać zadaniami.

### Wymagania:
- Node.js (zalecana wersja 16 lub nowsza)
- npm (Node Package Manager)

## 3. Testy (TaskManager_API.Test)

### Wymagania:
- Środowisko .NET 8.0 SDK

## Pełne uruchomienie aplikacji
 - `npm run start`
